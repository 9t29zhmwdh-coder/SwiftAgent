# Code of Conduct

## Pledge

I as the maintainer pledge to make participation in this project
a harassment-free experience for everyone, regardless of age, body size,
disability, ethnicity, gender identity and expression, level of experience,
nationality, personal appearance, race, religion, or sexual identity and orientation.

## Standards

Positive behavior: welcoming language, respect, graceful criticism acceptance.
Unacceptable: harassment, trolling, publishing private information without consent.

## Enforcement

Report via GitHub issue or maintainer profile. All reports reviewed promptly.
